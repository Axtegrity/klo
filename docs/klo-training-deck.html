KLO Platform 19
Admin

Countdown Clock & Display Settings

Countdown Clock

Found in the event edit form under the countdown toggle.

  • Toggle "Show Countdown Clock" on any upcoming event
  • Countdown shows months, days, hours, minutes, seconds
  • Appears at the top of the Events page, below the hero
  • Automatically disappears when the event date arrives
  • Multiple events can have countdowns simultaneously

Engagement Display Mode

Controls what name appears as the heading.

  • Event Name (default) — shows the conference/event name as the main heading
  • Session Name — shows the session name as the main heading on the engagement page and View More Details modal
  • Event cards on the events page always show event title first, session name second

Frequently Asked Questions

KLO Platform 21
FAQs

General Questions

Q: Do guests need to create an account to join a live session?
No. Guests can participate in polls, Q&A, and word clouds without signing in. They only need an account to download files or use the AI Advisor.
Q: How does the Engagement page know which event to show?
It checks today's date against all published events. If the event date matches today AND seminar mode is on, it appears. If only one event is live, it auto-selects.
Q: What happens to the countdown clock on event day?
It automatically disappears when the event date arrives. The event moves from the countdown section to the "Live Events" section with a "Join Event" button.
Q: Can I run multiple events on the same day?
Yes. The engagement page supports multiple simultaneous events. Guests will see a selection screen to pick which event to join.
KLO Platform 22
FAQs

Admin Questions

Q: Do I need to manually switch an event to "live"?
No. Events automatically show as live when today matches the event date. You only need to turn on Seminar Mode for the engagement page tools to work. The status override is only needed if you want to force a status change.
Q: What's the difference between Seminar Mode and the event status?
Seminar Mode controls whether the engagement page and "Engagement" nav link are visible at all. Event Status (upcoming/live/past) controls how the event appears on the events page. Both are independent — you need seminar mode ON for engagement tools to work.
Q: What's the max file size I can upload to an event?
50MB per file. Supported types: PDF, DOC, DOCX, PPT, PPTX, XLS, XLSX, TXT.
Q: How do I turn everything off after an event?
Turn off Seminar Mode in Admin → Conference tab. This hides the engagement page and removes the nav link. The event will automatically move to "Past Events" on the events page after its date passes.
KLO Platform 23
FAQs

Technical Questions

Q: Are session notes saved to a server?
No. Session notes are saved to the guest's browser (localStorage). They persist across page refreshes on the same device but are not backed up to a server. If the guest clears their browser data, notes are lost.
Q: How does the countdown handle time zones?
Each event has a timezone setting (default: Central Time). The countdown calculates the remaining time based on the event's timezone, not the viewer's local time. This ensures everyone sees the same countdown regardless of where they are.
Q: Is the mobile app the same as the website?
The mobile app wraps the same web app using Capacitor, adding native features: push notifications, biometric lock (Face ID/Touch ID), native share, and offline detection. The core experience is identical.
Q: Can guests see poll results after voting?
Yes. Results update in real time for everyone. After an event ends, closed poll results are also visible in the event's "View More Details" modal on the events page.
